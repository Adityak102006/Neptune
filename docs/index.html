How It Works

Three simple steps to
find any image

01

Index Your Folder

Point Neptune at a folder. It scans all images and extracts 1280-dimensional feature vectors using MobileNetV2.

02

Upload a Query

Upload a query image. Neptune computes its embedding and ranks all indexed images by cosine similarity.

03

View Results

Instantly see the top matches with similarity scores. Click on any result to view it full-size.
